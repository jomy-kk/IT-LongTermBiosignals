# IT-PreEpiSeizures
Computer Software towards the Denosing, Detection, Prediction and Forecasting of Epileptic Seizures. Based in Lisbon at Instituto de Telecomunicações (IT).

Developed and tested on Python 3.10.4. 

Requires:
* numpy v1.223
* mne v1.0.2
